# Cube Timer

Cube Timer is a web application where users can time their cube solves, view their solve history, and see a leaderboard of the fastest solvers.

## Features

- Start and stop the timer using the **spacebar**  
- View all previous solve times  
- Leaderboard of the fastest solvers  

## Technologies Used

- **Backend:** Flask  
- **Frontend:** React  
- **Database:** PostgreSQL  

## Installation

### Backend

1. Create and activate a virtual environment:
   
   ```bash
   python -m venv venv
   source venv/bin/activate   # On Windows: venv\Scripts\activate

2. Install dependencies:
   
   ```bash
   pip install -r requirements.txt

3. Run the Flask Server:
   ```bash
   flask run


### Frontend

1. Navigate to the frontend directory and install dependencies:

   ```bash
    npm install

2. Start the React development server:

   ```bash
    npm start


## Usage

- Open the app in your browser
- Press the spacebar to start/stop the timer
- View your solve history
- Check the leaderboard for the fastest solves
